# 🥗 Landing Page - NutriViva (Nutricionista)

Projeto acadêmico de **landing page responsiva** feito com **HTML + CSS puro** utilizando **Flexbox**.

## 🎯 Objetivo
Criar uma página única (landing page) para um pequeno negócio/autônomo (nutricionista), com seções sobre serviços, benefícios, depoimentos e formulário de contato, funcionando no desktop e no celular.

## 💻 Tecnologias usadas
- HTML5
- CSS3
- Flexbox
- Media Queries (responsividade)

## 📁 Estrutura do projeto
- `index.html` (estrutura da página)
- `style.css` (estilos e responsividade)

## 📌 Seções da página
- Topo (apresentação e chamada para ação)
- Sobre mim
- Serviços
- Benefícios
- Depoimentos
- Perguntas frequentes
- Contato (formulário)

## ▶️ Como executar
1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` no navegador

## 🌐 Publicação
O site foi publicado em hospedagem gratuita.

Link: https://pti-web-standards-beryl.vercel.app/

## 📝 Observações
O formulário é apenas demonstrativo (sem envio real), pois o foco do trabalho é HTML/CSS e layout responsivo.
